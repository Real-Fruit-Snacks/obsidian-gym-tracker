Features

Track your gains in Obsidian.

Detailed Logging

Log strength workouts with exercises, sets, reps, weight, RPE, RIR, completion state, and notes.

Ruck Tracking & GPX

Track rucks with distance, duration, and pack weight. Import GPX files to automatically fill ruck data.

Insights & Calendar

Review weekly volume, workout totals, streaks, exercise progress, and personal records. Browse workouts by month.

Workflow

Track. Review. Export.

1

Log Workouts

Build a strength or ruck draft, load routines, import GPX files, run the rest timer, and save the session.

2

Review Insights

Review totals, weekly trends, exercise progress, progression suggestions, and personal records.

3

Export as Markdown

Export workouts to Dataview-friendly Markdown with YAML frontmatter to perfectly integrate with your vault.

Get Started

Install in under a minute.

Recommended

Community Plugins

  1. Open Obsidian Settings and navigate to Community Plugins.
  2. Browse and search for Gym Tracker.
  3. Install and Enable the plugin.

Manual

  1. Download main.js, manifest.json, and styles.css from the latest release.
  2. Create .obsidian/plugins/gym-tracker/ inside your vault.
  3. Place the files inside that folder and reload Obsidian.
